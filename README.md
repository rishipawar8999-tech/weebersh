# Weeber Shopify Connector (weebersh)

This is the Weeber Shopify connector. It is an unlisted app whose ONLY purpose is OAuth — when a merchant installs it, their Shopify store gets connected to the Weeber platform.

## Installation

Merchants install the app via the following URL (generated by the Weeber dashboard):
`https://weebersh.myshopify.com/admin/oauth/authorize?client_id=...`

## How it works

After installation, the Weeber backend receives the access token via a secure callback. The merchant's store is then active in Weeber.
The app does not contain any embedded UI beyond a simple success/error screen to inform the merchant whether the connection was successful.

## Required Environment Variables

To run the app or deploy it, the following environment variables are required:
- `SHOPIFY_API_KEY`: The API key of the Shopify App
- `SHOPIFY_API_SECRET`: The API secret of the Shopify App
- `SHOPIFY_APP_URL`: The URL where this app is hosted
